// OVERVIEW SECTION STARTS

[float]
=== Descrição
Encerra a emulação de um teclado USB conectado a um computador. Para iniciar a emulação de um teclado, use link:../keyboardbegin[Keyboard.begin()].
[%hardbreaks]


[float]
=== Sintaxe
`Keyboard.end()`


[float]
=== Parâmetros
Nenhum

[float]
=== Retorna
Nada
