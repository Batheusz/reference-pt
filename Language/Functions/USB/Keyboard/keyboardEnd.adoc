// OVERVIEW SECTION STARTS
[#overview]

[float]
=== Descrição
Stops the keyboard emulation to a connected computer. To start keyboard emulation, use link:../keyboardbegin[Keyboard.begin()].
[%hardbreaks]


[float]
=== Sintaxe
`Keyboard.end()`


[float]
=== Parâmetros
Nothing

[float]
=== Retorna
Nothing
