= Mouse.begin()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrição
Begins emulating the mouse connected to a computer. `begin()` must be called before controlling the computer. To end control, use link:../mouseend[Mouse.end()].
[%hardbreaks]


[float]
=== Sintaxe
`Mouse.begin()`


[float]
=== Parâmetros
Nothing

[float]
=== Retorna
Nothing
