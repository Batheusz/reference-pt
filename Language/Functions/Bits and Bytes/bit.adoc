= bit()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrição
Computa o valor do bit especificado (o bit 0 é igual a 1, bit 1 igual a 2, bit 2 igual a 4, etc.).
[%hardbreaks]


[float]
=== Sintaxe
`bit(n)`


[float]
=== Parâmetros
`n`: o bit o qual se deseja computar o valor

[float]
=== Retorna
O valor do bit.
