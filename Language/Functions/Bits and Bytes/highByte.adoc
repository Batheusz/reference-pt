= highByte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrição
Extrai o byte mais significativo (o mais à esquerda) de uma word (valor de 16 bits), ou o segundo byte menos significativo de um tipo de dado maior.
[%hardbreaks]


[float]
=== Sintaxe
`highByte(x)`


[float]
=== Parâmetros
`x`: um valor de qualquer tipo

[float]
=== Retorna
byte
