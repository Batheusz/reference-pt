= lowByte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrição
Extrai o byte menos significativo (mais à direita) de uma variável (ex. uma word).
[%hardbreaks]


[float]
=== Sintaxe
`lowByte(x)`


[float]
=== Parâmetros
`x`: um valor de qualquer tipo

[float]
=== Retorna
byte
