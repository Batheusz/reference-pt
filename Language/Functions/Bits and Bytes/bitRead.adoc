= bitRead()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrição
Lê o valor de um bit em um número.
[%hardbreaks]


[float]
=== Sintaxe
`bitRead(x, n)`


[float]
=== Parâmetros
`x`: o número do qual se quer ler o bit

`n`: qual bit ler, começando em 0 para o bit menos significativo (mais à direita)


[float]
=== Retorna
O valor do bit (0 ou 1).
