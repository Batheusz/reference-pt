= sqrt(x)

// OVERVIEW SECTION STARTS

[#overview]
--

[float]
=== Descrição
Calcula a raiz quadrada de um número.
[%hardbreaks]


[float]
=== Sintaxe
`sqrt(x)`


[float]
=== Parâmetros
`x`: o número a se calcular a raiz quadrada, qualquer tipo de dados

[float]
=== Retorna
A raiz quadrada do número. (double)
