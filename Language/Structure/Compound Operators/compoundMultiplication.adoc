= *= Compound Multiplication


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrição
This is a convenient shorthand to perform multiplication of a variable with another constant or variable.
[%hardbreaks]


[float]
=== Sintaxe
[source,arduino]
----
x *= y;   // equivalent to the expression x = x * y;
----

[float]
=== Parâmetros
`x`: variável. *Tipos de dados permitidos:* int, float, double, byte, short, long +
`y`: variável ou constante. *Tipos de dados permitidos:* int, float, double, byte, short, long

--
// OVERVIEW SECTION ENDS



// HOW TO USE SECTION STARTS
[#howtouse]
--

[float]
=== Código de Exemplo

[source,arduino]
----
x = 2;
x *= 2;      // x now contains 4
----
