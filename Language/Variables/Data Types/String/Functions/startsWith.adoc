= startsWith()

// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrição
Testa se uma String começa ou não com os caracteres de uma outra String.

[%hardbreaks]


[float]
=== Sintaxe
[source,arduino]
----
string.startsWith(string2)
----

[float]
=== Parâmetros
`string, string2`: variáveis do tipo String


[float]
=== Retorna
`true`: se string começa com os carcateres de string2

`false`: do contrário
