= charAt()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrição
Access a particular character of the String.

[%hardbreaks]


[float]
=== Sintaxe
[source,arduino]
----
string.charAt(n)
----

[float]
=== Parâmetros
`string`: a variable of type String

`n`: a variable of type unsigned int


[float]
=== Retorna
The n'th character of the String.
